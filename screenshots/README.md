# Screenshots

Screenshots in this folder are referenced in the main README to give a quick visual preview of each tool.

## How to add a screenshot

1. Open the tool in your browser or desktop app
2. Take a screenshot (Windows: `Win + Shift + S`, Mac: `Cmd + Shift + 4`)
3. Crop to show the most useful part — don't screenshot your whole desktop
4. Save as **PNG**, max width **1280px** (resize if larger)
5. Name it to match what's referenced in README.md (e.g. `azure-cli.png`)
6. Commit it: `git add screenshots/your-tool.png && git commit -m "screenshots: add azure-cli"`

## Tips for good screenshots

- Use a clean browser profile (hide bookmarks bar, use private window)
- Dark mode or light mode — pick one and stick to it
- Blur or redact any subscription IDs, tenant IDs, or personal data before committing
- Add a red box or arrow annotation if you want to highlight something specific (use Snagit, Greenshot, or Paint)
